# دليل الإعداد السريع - GitHub Codespace

## 📋 ملخص سريع

هذا الدليل يشرح كيفية تفعيل موقع عيد المرأة في GitHub Codespace بخطوات بسيطة وسهلة.

---

## 🎯 الخطوات الأساسية (5 دقائق)

### 1️⃣ إنشاء مستودع GitHub

```
1. اذهب إلى github.com
2. اضغط على "+" → "New repository"
3. اسم المستودع: march8-celebration
4. اختر "Public"
5. اضغط "Create repository"
```

### 2️⃣ فتح Codespace

```
1. اضغط الزر الأخضر "Code"
2. اختر "Codespaces"
3. اضغط "Create codespace on main"
4. انتظر فتح المحرر (30 ثانية تقريباً)
```

### 3️⃣ إضافة الملفات

**الطريقة الأولى: النسخ واللصق (الأسهل)**

```
أ. انقر على أيقونة الملف الجديد (+)
ب. اكتب: index.html
ج. انسخ محتوى index.html من الأسفل
د. الصق المحتوى
ه. اضغط Ctrl+S للحفظ
و. كرر نفس الخطوات لـ styles.css و script.js
```

**الطريقة الثانية: السحب والإفلات**

```
أ. افتح مجلد المشروع على جهازك
ب. اسحب الملفات إلى نافذة Codespace
ج. أفلتها في الشريط الجانبي
```

### 4️⃣ إضافة الصور والـ GIF

```
1. اضغط على أيقونة "Upload" في الشريط الجانبي
2. اختر flowers.png
3. اختر givingflower.gif
4. انتظر انتهاء التحميل
```

### 5️⃣ تشغيل الموقع

```
1. اضغط Ctrl+` لفتح Terminal
2. اكتب: python3 -m http.server 8000
3. اضغط Enter
4. انقر على "Open in Browser"
5. استمتع بالموقع! 🎉
```

## 🔄 رفع التغييرات إلى GitHub

### الطريقة السهلة (3 خطوات)

```
1. اضغط على أيقونة "Source Control" (الفرع)
2. اضغط "+" بجانب "Changes" لإضافة الكل
3. اكتب رسالة: "Initial commit"
4. اضغط "Commit"
5. اضغط "Sync Changes"
```

---

## 🌐 نشر الموقع على الإنترنت

### GitHub Pages (مجاني وسهل)

```
1. اذهب إلى Settings في المستودع
2. اختر "Pages" من الجانب الأيسر
3. تحت "Source": اختر "Deploy from a branch"
4. اختر "main" و "/" (root)
5. اضغط "Save"
6. انتظر دقيقة واحدة
7. رابطك سيكون: https://yourusername.github.io/march8-celebration/
```

## 🎨 تخصيص الموقع

### تغيير الرسالة

**في index.html، ابحث عن:**
```html
<p class="message" id="message">
    بكل حب وتقدير، نتمنى لك يوماً مليئاً بالفرح والسعادة في هذا اليوم الخاص بك.
</p>
```

**غيّر النص إلى رسالتك الخاصة**

### تغيير الألوان

**في styles.css، ابحث عن:**
```css
:root {
    --color-pink: #FFB6C1;      /* اللون الوردي */
    --color-orange: #FFB347;    /* اللون البرتقالي */
}
```

**استخدم أكواد ألوان مختلفة من:**
- [Color Picker](https://htmlcolorcodes.com/)
- [Coolors](https://coolors.co/)
