# موقع عيد المرأة - С праздником 8 Марта

موقع احتفالي بسيط وجذاب بصفحتين، مصمم للاحتفال بعيد المرأة العالمي (8 مارس) بأسلوب عصري وتفاعلي.

## 🎨 الميزات الرئيسية

### الصفحة الأولى: التحية
- عنوان رئيسي أنيق بالروسية "С праздником" (بكل حب وتقدير)
- زر تفاعلي على شكل باقة ورد
- حركات ناعمة وتأثيرات بصرية جميلة
- جزيئات احتفالية تسقط برفق في الخلفية

### الصفحة الثانية: الرسالة والورد
- رسالة تهنئة شخصية تظهر بتأثير الكتابة (Typewriter Effect)
- GIF متحرك لهدية الورد بحركة عائمة ناعمة
- زر العودة البسيط والأنيق
- انتقال سلس بين الصفحات

## 🚀 كيفية التفعيل في GitHub Codespace

### الخطوة 1: إنشاء مستودع جديد على GitHub

1. اذهب إلى [GitHub.com](https://github.com)
2. انقر على الزر "+" في الزاوية العلوية اليمنى
3. اختر "New repository"
4. أدخل اسم المستودع: `march8-celebration`
5. اختر "Public" إذا أردت أن يكون الموقع متاحاً للجميع
6. انقر على "Create repository"

### الخطوة 2: فتح GitHub Codespace

1. في صفحة المستودع الجديد، انقر على الزر الأخضر "Code"
2. اختر التبويب "Codespaces"
3. انقر على "Create codespace on main"
4. سيتم فتح محرر VS Code في المتصفح

### الخطوة 3: إضافة الملفات

1. **انسخ محتوى الملفات التالية إلى Codespace:**

   **ملف index.html:**
   - انقر على أيقونة الملف الجديد في الشريط الجانبي
   - أنشئ ملف جديد باسم `index.html`
   - انسخ محتوى `index.html` من الأسفل والصقه

   **ملف styles.css:**
   - أنشئ ملف جديد باسم `styles.css`
   - انسخ محتوى `styles.css` من الأسفل والصقه

   **ملف script.js:**
   - أنشئ ملف جديد باسم `script.js`
   - انسخ محتوى `script.js` من الأسفل والصقه

2. **أضف الملفات الوسائط:**
   - انقر على أيقونة "Upload" في الشريط الجانبي
   - اختر `flowers.png` من الملفات التي أرسلتها
   - اختر `givingflower.gif` من الملفات التي أرسلتها

### الخطوة 4: معاينة الموقع

1. افتح Terminal في Codespace (Ctrl + `)
2. اكتب الأمر التالي:
   ```bash
   python3 -m http.server 8000
   ```
3. سيظهر إشعار أزرق بأسفل الشاشة
4. انقر على "Open in Browser" أو انسخ الرابط وافتحه
5. يجب أن ترى الموقع يعمل بنجاح!

### الخطوة 5: رفع التغييرات إلى GitHub

1. في الشريط الجانبي، انقر على أيقونة "Source Control" (فرع Git)
2. ستجد جميع الملفات تحت "Changes"
3. انقر على "+" بجانب كل ملف لإضافته (أو انقر على "+" بجانب "Changes" لإضافة الكل)
4. اكتب رسالة في حقل "Message" مثل: `Initial commit: Add March 8 celebration website`
5. انقر على الزر "Commit" (علامة الاختيار)
6. انقر على "Sync Changes" لرفع الملفات إلى GitHub

### الخطوة 6: نشر الموقع على GitHub Pages

1. اذهب إلى إعدادات المستودع (Settings)
2. اختر "Pages" من الشريط الجانبي الأيسر
3. تحت "Source"، اختر "Deploy from a branch"
4. اختر الفرع "main" والمجلد "/ (root)"
5. انقر على "Save"
6. بعد دقائق قليلة، سيظهر رابط الموقع الخاص بك في أعلى الصفحة
7. مثال: `https://yourusername.github.io/march8-celebration/`

## 🔧 التخصيص

### تغيير الرسالة الشخصية

افتح ملف `index.html` وابحث عن هذا السطر:

```html
<p class="message" id="message">
    بكل حب وتقدير، نتمنى لك يوماً مليئاً بالفرح والسعادة في هذا اليوم الخاص بك.
</p>
```

استبدل النص بالرسالة التي تريدها.

### تغيير الألوان

افتح ملف `styles.css` وابحث عن هذا القسم:

```css
:root {
    --color-pink: #FFB6C1;
    --color-orange: #FFB347;
    --color-white: #FFFFFF;
    --color-dark: #4A4A4A;
    --color-light-pink: #FFE4E1;
}
```

غيّر الأكواد السادسة عشرية للألوان التي تريدها.

### تغيير السرعات والحركات

في ملف `styles.css`، ابحث عن `animation-duration` و `animation-delay` لتغيير سرعات الحركات.

## 🎵 إضافة الموسيقى (اختياري)

إذا أردت إضافة موسيقى خلفية:

1. ضع ملف الموسيقى (مثل `themusic.mp3`) في نفس المجلد
2. أضف هذا السطر في نهاية `index.html` قبل `</body>`:

```html
<audio autoplay loop>
    <source src="themusic.mp3" type="audio/mpeg">
</audio>
```
